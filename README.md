# Daraz-Annual-Sales-Dashboard-Excel-Data-Analysis
This project presents an interactive annual sales dashboard built using Microsoft Excel to analyze business performance of Daraz.
The dashboard transforms raw sales data into meaningful visual insights to help understand trends, product performance, and revenue distribution.

🎯 Objectives

1.Analyze annual sales performance
2.Identify top-performing product categories
3Understand monthly sales trends
4.Compare regional sales performance
5.Create an interactive and easy-to-understand dashboard

🛠 Tools & Technologies

1.Microsoft Excel
2.Pivot Tables
3.Pivot Charts
4.Data Cleaning
5.Data Visualization

📊 Dashboard Features

📈 Monthly sales trend analysis

🛍 Category-wise sales performance

🌍 Region-wise revenue comparison

📊 Interactive charts and filters

📉 Clear visual representation of key business metrics
